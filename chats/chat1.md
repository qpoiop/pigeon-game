완성 — **《PIGEON PROTOCOL — 비둘기 특무》** (`Pigeon Protocol.dc.html`)

- **탑다운 3D 잠입 전략**: 전서구 요원이 마이크로필름 회수 → 적색 구역 탈출, 3스테이지 난이도 상승
- **자연스러운 절차적 애니메이션**: 비둘기 특유의 머리 까딱임, 발걸음, 웅크리기, 유휴 시 모이 쪼기
- **경비 AI**: 시야콘 + 가시선 판정, 감지 게이지, 추격/복귀. 웅크리기·은폐 구역으로 회피
- **조작**: WASD/화살표 + 바닥 클릭 + 모바일 조이스틱·버튼 (대시/숨기)
- **온라인**: 타이틀에서 방 코드 입력 시 무료 공개 릴레이(socketsbay)로 같은 방 요원이 서로 보임 — 공개 데모 채널이라 불안정하거나 안 될 수 있고, 그 경우 자동 싱글 진행
- **사운드**: WebAudio 미니멀 신스 (픽업/경보/클리어)
- Modernist 팔레트 그대로: 화이트 그리드 바닥, 잉크 벽, 레드 목표물, 플랫 HUD
